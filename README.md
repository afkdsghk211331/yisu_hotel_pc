# 易宿酒店预订平台 - 管理端（PC）

本项目为“易宿酒店预订平台”的 **PC 管理端**，面向 **商户/管理员** 两类角色：

- 商户：酒店信息录入、编辑、保存、提交审核
- 管理员：酒店信息审核（通过/驳回原因）、发布、下线、恢复

前端技术栈：**React 19 + TypeScript + TailwindCSS + Zustand + Vite**

---

## 功能范围

### 账号体系

- 注册：支持选择角色（商户 / 管理员）
- 登录：无需选择角色，登录后根据账号角色进入对应后台

### 商户端

- 酒店信息录入/编辑/修改（包含：中/英名、地址、星级、开业时间、房型、价格、图片等）
- 保存草稿/保存更新
- 提交审核（状态进入审核中）

### 管理员端

- 酒店列表按状态筛选：审核中/通过/不通过/已发布/已下线
- 审核：通过 / 驳回（驳回必须填写原因）
- 发布 / 下线（下线为状态变更，可恢复）

---

## 环境要求

- Node.js 18+（建议 20+）
- pnpm 8+

---

## 本地启动

安装依赖：

```bash
pnpm i
启动开发：

pnpm dev
构建：

pnpm build
预览构建产物：

pnpm preview
代码规范（ESLint / Prettier / Husky）
ESLint：静态检查（TS + React）

Prettier：统一格式化（含 Tailwind class 自动排序）

Husky + lint-staged：提交前自动修复与格式化

常用命令：

pnpm lint
pnpm lint:fix
pnpm format
pnpm format:check

备注
本项目为课程/作业性质实现，重点体现前端工程化、组件抽象、状态管理与流程设计。
```
